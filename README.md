# ocenum

Enum class for Objective-C.

## System Requirements

* Clang or GCC with Objective-C support
* Cocoa or GNUstep

Tested against GNUstep on Ubuntu 18.04 LTS. It should work on MacOS as well.

## Usage

Build a dynamic library:

```
$ make dynamic
```

Build a static library:

```
$ make static
```
